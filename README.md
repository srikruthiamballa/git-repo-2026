# 🚀 Git & GitHub Real World Training

Welcome to the Git Training Repository! This repository is designed to simulate a real-world project environment. Through a series of interactive tasks, you will learn essential Git skills that are used in daily software engineering, from opening your first Pull Request to untangling complex merge conflicts.

## 🎯 Modules

We have broken down the training into bite-sized modules:

- [Module 1: The First PR](./modules/01-THE-FIRST-PR.md)
- [Module 2: Branching & Merging](./modules/02-BRANCHING-MERGING.md)
- [Module 3: Conflict Resolution](./modules/03-CONFLICT-RESOLUTION.md)
- [Module 4: Commit Squashing (Clean History)](./modules/04-SQUASHING.md)
- [Module 5: Time Travel (Reverts & Resets)](./modules/05-TIME-TRAVEL.md)
- [Module 6: Stashing & Ignoring Files](./modules/06-STASHING-AND-IGNORE.md)
- [Module 7: Syncing Your Fork](./modules/07-SYNCING-FORKS.md)

## 🛠 Getting Started

Head over to [CONTRIBUTING.md](./CONTRIBUTING.md) for instructions on how to fork, clone, and set up this repository locally. Once done, start with Module 1!
